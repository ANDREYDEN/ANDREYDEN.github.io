GitHub | Try it Out

Background

This is a small project inspired by one of Google's doodles. Logeomio is a tool for representing words (logos) as a collection of geometrical figures.

Each time a logo is generated, the canvas is split into thousands of different polygons (shards). After that each shard is filled either black or white depending on whether it contains a letter or not.

What's inside?

HTML, CSS and lots of p5.js
