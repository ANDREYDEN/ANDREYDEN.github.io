Game Mechanics

This game is a classic maze solver with a twist: the character is holding a torch that is slowly running out making the lit maze area smaller with every second. Once the torch is out completely, the character is deemed lost in the darkness. More complicated levels include locating keys to unlock the exit and avoiding ghosts that are wandering around the maze.

Technical Challenges

There were 2 particularly interesting technical problems to solve during the development of the game:

  1. Maze Generation. There are a lot of different ways to create mazes, so a few generation strategies are available in the game using variations of the DFS and BFS algorithms.
  2. Level Replay. One of the nice features in Spooky Maze is the ability to replay the level both forwards and backwards in time. This required storing all the individual moves of the character (as well as the moves of the NPCs) to replay them in either direction.

Background

Spooky Maze was created as part of my work for the Maple Mana game studio. The game was supposed to be released for Halloween, hence the spooky theme.
