Background

Another team, another year, same competition - it's Space Apps 2019.

This time we came up with an interesting way of showing real-time satellite data. Each satellite is represented as an emoji, the type of the emoji depends on the kind of data that satellite is collecting. For example, the METEOR PRIRODA satellite monitors atmospheric and sea-surface temperatures, so it is represented as a thermometer emoji.

This project is still in development. More information about the future of the project can be found on GitHub.
