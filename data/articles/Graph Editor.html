Background

At high school I was taught a lot of back-end logic and algorithms (which, don't get me wrong, I appreciate and am grateful my teachers for). But there was always a desire not just to operate the numbers, but also to display the data in a meaningfull way. And it was not until Grade 11 when I discovered The Coding Train - a YouTube channel full of different interesting programming concepts and visualizations made by using this tool called Processing (later p5.js).

So when we had to do some visual programming at the end of high school I decided to create something using Processing.

All the functionality description as well as download instructions can be found on the GitHub page of the project.
